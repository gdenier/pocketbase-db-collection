# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-08

### Added
- Initial release
- PocketBase collection options creator for TanStack DB
- Bidirectional sync with PocketBase using Server-Sent Events
- Support for parse/serialize data transformations
- ID tracking to prevent duplicate operations
- Configurable mutation timeout
- Initial fetch options (sort, filter, expand)
- Comprehensive error types
- Full TypeScript support
